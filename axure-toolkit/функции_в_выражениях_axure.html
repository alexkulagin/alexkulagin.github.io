Пример демонстрирует возможность использования функций в выражениях Axure. В примере происходит регистрация пользовательской функции "MyExpression" с помощью импорта сценария. Функция возвращает сумму аргументов. Вызов функции происходит из действия "Set Text".

Пример использования функции в выражении

javascript:

    $m.addExpression("MyFunction", function(a, b) {

        return Number(a) + Number(b);

    });

Следующий пример сценария демонстрирует регистрацию пользовательской функции, которую можно вызвать из выражения прототипа Axure.

Пример регистрации

Вызвать метод "addExpression" можно двумя способами, с помощью импорта JavaScript сценария или вызвать подготовленный сценарий действием "Открыть ссылку в новом окне" (Open Link in New Window/Tab).

$m.addExpression(name, function);

Чтобы добавить пользовательскую функцию, необходимо вызвать специальный публичный метод "addExpression" глобального объекта $m. Этот метод отвечает за регистрацию новых пользовательских функций. Метод принимает два аргумента — строковое название и функцию обратного вызова (callback).

Как добавить пользовательскую функцию

1. Находит пользовательскую функцию "example".

2. Вызывает эту функцию с аргументами arg1 и arg2.

3. Возвращает значение функции, например результат вычисления из аргументов.

[["example".trim(arg1, arg2)]]

1. Удаляет пробельные символы в начале и в конце строки " example ".

2. Возвращает "example".


Но стоит добавить хотя бы один аргумент и поведение функции "trim" меняется:

[[" example ".trim()]]

По умолчанию функция "trim" запускается без аргументов, она убирает лишние пробельные символы с начала и конца строки:

Как это работает

В Axure RP можно использовать выражения обрамленные двойными квадратными скобками

— "[[expression]]". По умолчанию внутри выражений вызываются предустановленные функции для работы с виджетами, строками, числами, датами, логическими операторами, локальными или глобальными переменными и прочее.


Выражения могут использоваться практически со всеми возможными действиями Axure RP, но к сожалению, внутри выражений невозможно вызвать пользовательскую функцию или JavaScript сценарий.


Axure Toolkit предоставляет отличный способ для внедрения пользовательских функций в выражения, основанный на переопределении функции "trim", которая используется для обработки строковых литералов.
