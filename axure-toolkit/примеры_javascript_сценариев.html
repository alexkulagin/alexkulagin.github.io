Пример работы с браузерной консолью

После нажатия в браузерной консоли появляется сообщение "Hello!".

Пример использования локальной переменной в сценарии

Демонстрация возможности использования локальных переменных прототипа в сценарии. После нажатия в браузерной консоли появляется сообщение "Button text: [[текст кнопки]]" и "Button name: [[название кнопки]]".

Пример использования jQuery

В примере продемонстрирована возможность управления виджетами с помощью jQuery. Переключатель включает режим редактирования и позволяет вводить текст непосредственно в кнопки.

Дополнительный пример jQuery

В этом примере переключатель включает режим обрезания длинной строки заголовка средствами css до 150px.
