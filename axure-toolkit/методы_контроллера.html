Возможности Dynamic Panel

/* находит и возвращает виджеты WidgetA из состояний с индексами 2 и 5 */

var widgets = controller.getWidget('WidgetA', [2, 5]);

Метод "getWidget" позволяет находить виджеты из списка состояний.

/* находит и возвращает виджеты WidgetA и WidgetB из состояния с индексом 3 */

var widgets = controller.getWidget(['WidgetA', 'WidgetB'], 3);

Метод "getWidget" позволяет находить виджеты по списку названий.

/* находит и возвращает виджет с названием WidgetA из состояния с индексом 1 */

var widget1 = controller.getWidget('WidgetA', 1);


/* находит и возвращает виджет WidgetB из состояния с названием State1 */

var widget2 = controller.getWidget('WidgetB', 'State1');

Возвращает по имени виджет из конкретного состояния динамической панели.

Метод getWidget

Из полезных свойств объекта текущего состояния можно выделить следующие:

• index — индекс текущего состояния.

• label — название текущего состояния.

/* сохраняет в переменную объект текущего состояния */

var current = controller.state();


/* полезные свойства объекта текущего состояния */

var index = current.index;

var label = current.label;

Если метод "state" вызван без параметров, то он возвращает объект, описывающий текущее состояние виджета "Dynamic Panel".

/* меняет состояние по индексу следующего состояния */

controller.state(2);


/* меняет состояние по названию следующего состояния */

controller.state('State2');

Меняет состояние виджета "Dynamic Panel". В качестве параметра методу передается название или индекс следующего состояния.

Метод state

В интерфейсах контроллера присутствуют два полезных метода — первый "state" меняет или возвращает текущее состояние, второй "getWidget", находит и возвращает виджет из конкретного состояния.
