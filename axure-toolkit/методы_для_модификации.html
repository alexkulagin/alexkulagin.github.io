Возможности Repeater

/* записывает в переменные строки */

var rowsA = controller.get([2, 3, 4]);

var rowsB = controller.get([1, 5, 6]);


/* применяет сначала строки rowsA */

controller.applyRows(rowsA).render();


/* затем применяет строки rowsB */

controller.applyRows(rowsB).render();

Удаляет все строки из модели и применяет новые.

Метод applyRows

/* восстанавливает первоначальное состояние */

controller.recover().render();

Восстанавливает первоначальные данные модели.

Метод recover

/* данные, выступающие в качестве обновления */

var data = { name: 'admin', status: 2 };


/* обновляет ячейки всех строк репитера */

controller.update(data).render();


/* обновляет ячейки второй строки */

controller.update(data, 2).render();


/* обновляет ячейки второй, четвертой и пятой строки */

controller.update(data, [2, 4, 5]).render();


/* обновляет строки со значением admin_1 в ячейке name */

var filterA = { condition: { name: 'admin_1' } };

controller.update(data, filterA).render();

Обновляет ячейки в строках модели по условию. В качестве условия может передаваться индекс строки, список индексов или фильтр.
