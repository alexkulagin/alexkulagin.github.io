Демонстрация запуска цепочки сценариев с помощью нового способа. Нажатие на кнопку отображает в браузерной консоли сообщения об успешном вызове.

Пример запуска цепочки сценариев

По умолчанию недокументированная возможность запуска JavaScript сценариев в действии "Открыть ссылку в текущем окне" (Open Link in Current Window) не позволяет запускать подряд несколько сценариев. В результате выполнения "всегда" будет вызываться последний сценарий из списка.


Axure Toolkit предоставляет альтернативный способ вызова сценариев, с помощью действия "Открыть ссылку в новом окне" (Open Link in New Window/Tab). При этом, возможность открытия гипер ссылок в новом окне полностью сохраняется. Поведение действия меняется, только если адрес ссылки начинается с "javascript:".
