Все нижеследующие примеры запускаются в действии "Открыть ссылку в текущем окне" (Open Link in Current Window) без использования внешних расширений.

Примеры выполнения JavaScript сценариев

javascript:


    /* переменные */

    var a = 25;

    var b = 25;


    /* функция */

    function sum (argA, argB) {

        return argA + argB;

    }


    /* присвоение переменной "c" значения функции */

    var c = sum(a,b);


    /* выводит в браузерную консоль значение переменной "c" */

    console.log(c);


void 0;

В сценариях, добавленных с помощью "OpenLink" можно использовать практически все возможности языка JavaScript — переменные, функции, встроенные функции, массивы, циклы, функциональные выражения и прочее.

Возможности JavaScript

javascript:


    /* следующее выражение отобразит в браузерной консоли Hello world! */

    console.log('Hello world!');


void 0;

Для отладки JavaScript можно использовать браузерную консоль.

Отладка сценариев

javascript:


    /* переменная доступна в пределах этого сценария */

    var name = "abc";


    /* переменная доступна из других сценариев, но только в пределах текущей страницы */

    window.name = "abc";


void 0;

Переменные, определенные в сценариях через "var" доступны только в пределах выполняемого сценария. Чтобы обеспечить доступ к переменной из другого сценария, в пределах страницы ее необходимо определить в объекте "window".

Видимость переменных

javascript:


    // однострочный комментарий заблокирует выполнение, следующих за ним выражений


    /*

        многострочный

        комментарий

        не заблокирует

    */


void 0;

В сценариях можно использовать только многострочные комментарии "/* comment.. */"

Комментарии в сценариях

javascript:

    var a = 'name';

void 0;

Все JavaScript сценарии, добавленные в действие "OpenLink" должны обязательно начинаться со строки "javascript:" и заканчиваться строкой "void 0;"
