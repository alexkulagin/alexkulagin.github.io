Возможности Repeater

/* удаляет вторую строку */

controller.remove(2).render();


/* удаляет вторую, четвертую и пятую строку */

controller.remove([2, 4, 5]).render();


/* удаляет все строки кроме второй и четвертой */

controller.remove([2, 4, 5], true).render();


/* удаляет все строки со значением admin_1 в ячейке name */

var filterA = { condition: { name: 'admin_1' } };

controller.remove(filterA).render();

Удаляет строки из модели по условию. В качестве условия может передаваться индекс строки, список индексов или фильтр.

Метод remove

/* удаляет все строки и сразу запускает обновление HTML представления */

controller.clear().render();

Удаляет все строки модели.
