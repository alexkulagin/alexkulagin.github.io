Расширяет возможности Axure RP 8

AXURE TOOLKIT

0.0.0

Axure Toolkit

Набор инструментов для расширения возможностей прототипов Axure RP 8 с помощью простых JavaScript сценариев.

Требования

• Продвинутые навыки работы с Axure

• Базовые знания JavaScript (уровень "начинающий")

• Базовые знания HTML/CSS

• Современный браузер для демонстрации прототипа

Ключевые возможности Axure Toolkit

Активирует возможность запуска сценариев из действия "Открыть ссылку в новом окне" (Open Link in New Window/Tab).


Предоставляет способ для импорта и инициализации JavaScript сценариев до запуска внутреннего события "OnPageLoad".


Добавляет возможность вызова пользовательских функций из выражений прототипа. Выражения могут использоваться практически со всеми возможными действиями Axure RP.


Добавляет в прототип широковещательные сообщения, которые можно адресовать в любое место прототипа – в динамическую панель, в местер, в любой фрейм, где бы он не находился.


Расширяет возможности виджетов "Dynamic Panel" и "Repeater".

После вставки скрипта, расширение будет доступно на всех сгенерированных страницах прототипа, включая вложенные фреймы.

Как происходит внедрение расширения

Поля настройки "Include Viewport Tag" во время генерации прототипа не валидируются и передаются как есть. Вместе со значением метатега можно передать инъекцию с кодом — для этого нужно закрыть из поля значения текущий метатег, вставить сценарий и открыть метатег пустышку, примерно так:

no"/><script> Your JavaScript Injection </script><meta name="

Эта допущение позволяет расширению внедриться на все страницы прототипа и гарантированно выполнить скрипт до запуска любого сценария из Axure API.

В какой момент происходит активация расширения

Axure Toolkit во время выполнения подписывается на событие "DOMContentLoaded", которое срабатывает сразу после загрузки страницы, не дожидаясь загрузки изображений и стилей. В момент срабатывания из кода доступны все HTML представления прототипа и все JavaScript сценарии Axure API.
