Возможности Dynamic Panel

Контроллер сохраняется в переменной "controller" при условии, что виджет "Название виджета" присутствует на странице, его название уникально в пределах страницы и тип соответствует динамической панели. Иначе, в переменной будет сохранен null.


Если условия соблюдены, то можно обращаться к переменной "controller" и вызывать доступные методы контроллера.

javascript:

    

     /* находит виджет по названию */

     var widget = $a('@' + 'Название виджета');


     /* возвращает контроллер для виджета Dynamic Panel */

     var controller = widget.getPanelController();


void 0;

В следующем примере происходит поиск виджета, результат этого поиска сохраняется в переменной "widget". Далее метод "widget.getPanelController" возвращает контроллер и сохраняет его в переменной "controller".

Метод getPanelController

Axure Toolkit добавляет в объект виджета специальный метод "getPanelController", который создает и возвращает контроллер для конкретного виджета "Dynamic Panel".
