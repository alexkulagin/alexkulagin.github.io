Возможности выражений

Выражение запустит функцию, сложит переданные аргументы и вернет сумму (80).

[["MyFunction".trim(24, 56)]]

В примере демонстрируется вызов зарегистрированной функции "MyFunction".

Пример вызова функции

javascript:

    $m.addExpression("MyFunction", function(a, b) {

        return Number(a) + Number(b);

    });

void 0;

Следующий пример демонстрирует регистрацию пользовательской функции, которую можно вызвать из выражения прототипа Axure.

Пример регистрации

Вызвать метод "addExpression" можно двумя способами, с помощью импорта JavaScript сценария или вызвать подготовленный сценарий из редактора значений "fx".

$m.addExpression(name, function);

Чтобы добавить пользовательскую функцию, необходимо вызвать специальный публичный метод "addExpression" в глобальном объекте "$m". Этот метод отвечает за регистрацию новых пользовательских функций. Метод принимает два параметра — строковое название и функцию обратного вызова (callback).
