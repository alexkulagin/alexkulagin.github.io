Возможности Repeater

Фильтр "filterC" отфильтрует все строки, в которых присутствует хотя бы одно совпадение из перечисенных условий.

var filterC = { any: true, condition: { name: 'admin_1', type: 'super' } };

С помощью параметра "any" можно упростить поиск и позволить фильтровать любые совпадения из списка условий. Для этого, параметру "any" нужно присвоить true (по умолчанию присвоен false).

Параметр any

var filterB = { condition: { name: 'admin_1', type: 'super', status: 'red' } };

В следующем примере демонстрируется фильтр "filterB", под который попадают все строки, в которых ячейка "name" равна значению "admin_1", ячейка "type" равна "super" и "status" равен "red".

var filterA = { condition: { name: 'admin_1' } };

В примере демонстрируется фильтр "filterA", под который попадают все строки, в которых ячейка "name" равна значению "admin_1".

Примеры фильтров

Методы контроллера "applyFilter", "get", "copy", "update" и "remove" принимают два параметра "filter" и "exclude".


Первый параметр "filter" находит строки по определенному условию. Второй "exclude", превращает список найденых строк в исключение, т.е. инвертирует фильтрацию и возвращает все строки, кроме тех, что совпали с условием из параметра "filter".
