В следующем примере продемонстрирован способ группировки виджетов для импорта JavaScript сценариев с помощью динамической панели "ax.bundle". Пример полностью повторяет поведение из предыдущего примера "Пример импорта JavaScript сценариев".

Пример группировки виджетов для импорта

Виджеты "ax.import" с JavaScript сценариями можно группировать, для этого необходимо создать динамическую панель с названием, начинающимся с "ax.bundle" и поместить туда виджеты для импорта.


Чтобы обеспечить правильную зависимость, внутри панели можно настроить порядок импорта и выполнения. Для этого достаточно поменять местами виджеты в "Outline" — зависимые должны находиться выше, т.к. запуск и выполнение происходит последовательно снизу вверх.

Группировка импортируемых сценариев

В следующем примере продемонстрирован импорт двух пользовательских JavaScript сценариев. Результат выполнения и оповещение об успешном импорте выводится в браузерную консоль.

Пример импорта JavaScript сценариев

Axure Toolkit предоставляет простой способ для внедрения пользовательских сценариев и библиотек в проекты Axure RP 8. Для импорта необходимо создать виджет с названием, начинающимся с "ax.import" и вставить в него код сценария.


В качестве импортирующего виджета может выступать любой виджет, который поддерживает событие "OnMove", и в который можно вставить текст.


Для удобства виджеты можно именовать, например "Первая библиотека ax.import", "ax.import Функция обработки" или "ax.import.Утилиты" — главное, чтобы в названии обязательно присутствовал идентификатор "ax.import".


Сценарий расширения во время выполнения находит виджеты с названием, которое содержит "ax.import" и прожигает в них событие "OnMove". Несколько таких виджетов отработают последовательно снизу вверх, в порядке расположения в окне "Outline".


Импорт всегда происходит до инициализации сценариев. После выполнения все содержимое импортирующих виджетов автоматически очищается, а сам виджет скрывается (чтобы не засорять DOM).
