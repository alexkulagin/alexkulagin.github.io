Широковещательные сообщения

javascript:


     /* функция обработчик */

     function MyHandler (channel, message) {

          console.log(message);

     }


     /* регистрация обработчика */

     $m.listen('channel-name', ['widgetA', 'widgetB', MyHandler]);


void 0;

Регистрирует слушателя для сообщений канала "channel.name". В качестве обработчиков слушателя выступают два виджета и одна функция. В время обработки сообщения в виджетах с именами "widgetA" и "widgetB" будет вызвано событие "OnMove", после вызова будет выполнен последний обработчик — функция "MyHandler".

Пример регистрации списка обработчиков

javascript:


     /* функция обработчик */

     function MyHandler (channel, message) {

          console.log('канал:', channel);

          console.log('сообщение:', message);

     }


     /* регистрация обработчика */

     $m.listen('channel-name', MyHandler);


void 0;

Регистрирует слушателя для сообщений канала "channel.name". В качестве обработчика слушателя выступает функция "MyHandler".

Пример регистрации функции обработчика

javascript:

     $m.listen('channel.name', 'widget.name');

void 0;

Регистрирует слушателя для сообщений канала "channel.name". В данном примере виджет с названием "widget.name" будет обработчиком слушателя. В время обработки сообщения в виджете "widget.name" будет вызвано событие "OnMove".
