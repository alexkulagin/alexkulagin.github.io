Использование сценариев в Axure RP 8

AXURE JAVASCRIPT

Особенности использования сценариев

Все сценарии, добавленные в редактор "fx" должны обязательно начинаться со строки "javascript:" и заканчиваться строкой "void 0;".

javascript:

    var a = 'name';

void 0;

Комментарии в сценариях

В сценариях, которые были добавлены в редактор "fx" можно использовать только многострочные комментарии "/* comment.. */".

javascript:


    // однострочный комментарий заблокирует выполнение всех следующих за ним выражений


    /*

        многострочный

        комментарий

        не заблокирует

    */


void 0;

Видимость переменных

В редакторе "fx" переменные, определенные через "var" доступны только в пределах выполняемого сценария. Чтобы обеспечить доступ к переменной из другого сценария, ее необходимо определить в объекте "window".

javascript:


    /* переменная доступна только в пределах этого сценария */

    var name = "abc";


    /* переменная доступна из других сценариев, но только в пределах текущей страницы */

    window.name = "abc";


void 0;

Отладка сценариев

Для отладки JavaScript можно использовать браузерную консоль.

javascript:


    /* следующее выражение отобразит в браузерной консоли Hello world! */

    console.log('Hello world!');


void 0;

Возможности JavaScript

В сценариях, добавленных в редактор значений "fx" можно использовать практически все возможности языка JavaScript — переменные, функции, встроенные функции, массивы, циклы, функциональные выражения и прочее.

javascript:


    /* переменные */

    var a = 25;

    var b = 25;


    /* функция */

    function sum (argA, argB) {

        return argA + argB;

    }


    /* переменной "c" присваивается значение, возвращаемое функцией "sum" */

    var c = sum(12, 10);


    /* выводит в браузерную консоль значение переменной "c" */

    console.log(c);


void 0;
