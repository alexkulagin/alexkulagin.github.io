/* возвращает объект виджета с именем test из 4 строки */

var widget = controller.getWidget('test', 4);

    widget.text('edit..');


/* возвращает объект виджета с именем test из 2 и 6 строки */

var widgets = controller.getWidget('test', [2, 6]);

    widgets.text('edit..');


/* возвращает объект виджета с именем name или с именем type из 1 и 3 строки */

var widgets = controller.getWidget(['name', 'type'], [1, 3]);

    widgets.text('edit..');

Возвращает виджет по имени из конкретной строки модели.

Метод getWidget

/* вернет кол-во строк */

var length = controller.getLength();

Возвращает текущее количество строк в модели.

Метод getLength

/* суммирует все значения из столбца track и возвращает результат */

var sum = controller.getSum('track');

Возвращает сумму значений из конкретного столбца.

Метод getSum

/* вернет список значений из столбца name */

var names = controller.getColumn('name');

Возвращает массив значений из конкретного столбца.

Метод getColumn

/* вернет признак наличия столбца name */

var name = controller.hasColumn('name');

Возвращает признак наличия столбца в модели ("true" или "false").

Метод hasColumn

/* возвращает копию третьей строки */

var rows = controller.copy(3);


/* возвращает копию строк со значением "admin_1" в ячейке "name" */

var filterA = { condition: { name: 'admin_1' } };

var rows = controller.copy(filterA);

Метод работает так же как метод "get", только вместо исходных строк возвращает их точные копии. В качестве условия для поиска может передаваться индекс, список индексов или фильтр.

Метод copy

/* возвращает все строки */

var rows = controller.get();


/* возвращает третью строку */

var rows = controller.get(3);


/* возвращает вторую и третью строку */

var rows = controller.get([2, 3]);


/* возвращает все строки кроме второй и третьей */

var rows = controller.get([2, 3], true);


/* возвращает строки со значением admin_1 в ячейке name */

var filterA = { condition: { name: 'admin_1' } };

var rows = controller.get(filterA);


/* возвращает все строки кроме admin_1 или super */

var filterB = { any: true, condition: { name: 'admin_1', type: 'super' } };

var rows = controller.get(filterA, true);

Находит и возвращает строки по условию. В качестве условия может передаваться индекс строки, список индексов или фильтр.
